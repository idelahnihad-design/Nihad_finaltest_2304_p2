## Instructions
1. Clone the repository locally using `git clone <URL>`.

## Reflection
For this test I created the repository, set up the milestone 
"Nihad_ft_milestone_1" with a due date of April 30, 2026, and linked two 
issues to it: one for adding the Python server file and one for the HTML 
template. I assigned both issues to myself. The main challenge was 
remembering to link each issue to the milestone before submitting it, which 
I solved by using the sidebar on the issue creation page. Cloning the 
repository locally confirmed that all files were pushed correctly.
